🍗 Cluckin' Bell - San Andreas Tribute Landing Page Uma landing page temática e responsiva inspirada na icônica rede de fast food do jogo GTA San Andreas. O projeto simula um portal de ofertas onde os clientes (como CJ e Big Smoke) podem se cadastrar para receber promoções.

🚀 Funcionalidades e Destaques Menu Dinâmico: Listagem de produtos (Hambúrguer de Frango, Burrito de Salamandra, etc.) utilizando o sistema de colunas do Bootstrap.

Formulário de Leads: Seção de cadastro funcional com validação de campos.

Interatividade com JS: Script para manipulação de eventos (Submit), reset de formulário e feedback visual temporário para o usuário.

Design Temático: Uso de assets visuais do universo GTA para criar uma experiência imersiva.

Responsividade: Interface adaptável para diferentes tamanhos de tela através do Bootstrap 5.

🛠️ Tecnologias Utilizadas HTML5: Estrutura semântica.

CSS3: Estilização customizada (classes como .menu-item, .form-wrapper e .side-image).

Bootstrap 5.3: Utilizado para Grid System, espaçamentos (mb-3) e componentes de formulário.

JavaScript (Vanilla): Lógica para manipulação do DOM e interceptação do formulário.
